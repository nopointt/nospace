# Competitor Landscape

## Direct Competitors (Context Storage / RAG-as-a-Service)

### Ragie
- Positioning: "Context Engine for LLM apps"
- Funding: $5.5M seed
- Pricing: $250/month minimum (connector tax model)
- Weakness: expensive, developer-only, no non-tech UX

### Graphlit
- Positioning: "Context Layer for AI"
- Feature: knowledge graph extraction
- Weakness: zero public reviews, unclear adoption

### Morphik
- Positioning: YC X25 batch, ColPali visual retrieval
- License: BSL (not truly open source)
- Weakness: early stage, restrictive license

### Vectorize
- Positioning: "Agent Memory", RAG evaluation tools
- Note: pivoted from Hindsight (analytics)
- Weakness: evaluation focus, not end-user product

### Langbase Memory
- Positioning: Part of Langbase AI development platform
- Pricing: 50 MB at $250/month
- Weakness: very limited storage per dollar, ~6 formats only

### Supermemory
- Positioning: "Context infrastructure", open-core
- Pricing: $19 to $399 cliff
- Funding: $3M
- Weakness: dev-first, open-core trap (features locked behind paid tier)

## Contexter's White Space

The competitive landscape reveals a clear white space: **universal + non-technical**. All direct competitors target developers with technical APIs. None offer a simple "upload anything, ask anything" experience for non-technical users.

Contexter's positioning: "One memory. Every AI." — emphasizes universality (any format, any LLM) and simplicity (non-tech audience).

## Key Differentiators

1. **Format breadth:** 15 formats vs 5-8 for competitors
2. **MCP-native:** First-class MCP integration, not an afterthought
3. **Price:** $9/month starter vs $250+ for Ragie/Langbase
4. **Non-tech UX:** Upload and query from a simple web interface
5. **Privacy:** EU-hosted, no model training on user data
