Весь интерфейс Harkly — **на русском языке**. Аудитория: RU/CIS B2B (продакт-менеджеры, исследователи, дизайнеры).

**Why:** nopoint явно указал — интерфейс под ру СНГ аудиторию.

**How to apply:** Все Pencil-фреймы, лейблы, кнопки, подсказки, названия флоров, пилюли — только по-русски. Английские аббревиатуры/термины OK как proper nouns (PICOT, Harkly, F0–F5, SpineProgress).

**Переводы ключевых элементов:**
- Framing Studio → Студия фрейминга
- Start Research → Начать исследование
- Refine → Уточнить
- Scratchpad → Черновик (F0)
- Connectors → Источники (F1)
- Raw → Сырые данные (F2)
- Insights → Инсайты (F3)
- Artifacts → Артефакты (F4)
- Stakeholders → Стейкхолдеры (F5)
- Branch → Ветка
- Population → Популяция (PICOT P)
- Intervention → Воздействие (PICOT I)
- Comparison → Сравнение (PICOT C)
- Outcome → Результат (PICOT O)
- Time → Период (PICOT T)
