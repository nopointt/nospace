YOUR TASK: Create the Schema Confirmation UI and Extraction Results UI for Harkly MVP.

## Files to create:

### 1. src/routes/(protected)/kb/[kbId]/schema/index.tsx — Schemas list
- List of schemas for this KB
- Each item: name, version, status badge (draft/confirmed/archived)
- "Обнаружить схему" button → navigates to discover page
- "Создать из инструкции" button → shows inline text input
- Fetch from GET /api/kb/{kbId}/schemas
- Russian UI, light theme

### 2. src/routes/(protected)/kb/[kbId]/schema/discover.tsx — Schema discovery
- "Обнаружить схему" button triggers POST /api/kb/{kbId}/schema/discover
- Loading state: "AI анализирует документы..."
- On success: shows proposed SchemaField[] as editable list
- Each field row: name input, type dropdown (string/number/boolean/date/array/object/enum), required checkbox, description input
- User can: add field, remove field, edit any property
- "Подтвердить схему" button → POST /api/kb/{kbId}/schemas/{schemaId}/confirm
- Russian UI

### 3. src/routes/(protected)/kb/[kbId]/schema/[schemaId].tsx — Schema editor
- Load schema + fields from GET /api/kb/{kbId}/schemas/{schemaId}
- Same editable field list as discover page
- Save changes: PUT /api/kb/{kbId}/schemas/{schemaId} with updated fields
- If status=draft: show "Подтвердить" button
- If status=confirmed: show "Архивировать" option
- Russian UI

### 4. src/routes/(protected)/kb/[kbId]/extract.tsx — Run extraction
- Select a confirmed schema from dropdown (fetch from GET /api/kb/{kbId}/schemas?status=confirmed)
- "Запустить экстракцию" button → POST /api/kb/{kbId}/extract with { schemaId }
- Progress polling: "Обработано X из Y документов..."
- On complete: link to results page
- Russian UI

### 5. src/routes/(protected)/kb/[kbId]/extractions.tsx — Extraction results table
- Fetch from GET /api/kb/{kbId}/extractions
- Table with columns dynamically generated from schema fields
- Each row = one extracted entity
- Confidence column with color-coded badge:
  - confidence > 0.8 → green "Высокая"
  - confidence > 0.5 → yellow "Средняя"
  - confidence < 0.5 → red "Низкая"
- Click row → expand to show evidence (source text quote)
- "Экспорт CSV" button at top
- Russian UI

### 6. src/components/SchemaFieldEditor.tsx — Reusable schema field editor
- Props: fields: SchemaField[], onChange: (fields: SchemaField[]) => void
- Renders list of field editors
- Each field: name input, type select, required toggle, description textarea
- Type options: строка, число, логическое, дата, массив, объект, перечисление
- "Добавить поле" button at bottom
- "Удалить" button per field (trash icon)
- Drag handle for reorder (optional, or just up/down buttons)
- SchemaField type: { name: string, type: string, description?: string, required?: boolean, enumValues?: string[] }
