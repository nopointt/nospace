This is a SolidStart + SolidJS + Tailwind CSS 4 project deploying to Cloudflare Pages.

YOUR TASK: Create the Upload UI for Harkly MVP. Users upload files to their knowledge base.

## Files to create:

### 1. src/routes/(protected)/kb/[kbId]/upload.tsx — Upload page
- SolidJS component with:
  - File picker (click to browse) + drag-and-drop zone
  - Accepted file types: .pdf, .docx, .csv, .txt, .xlsx, .md, .json, .mp3, .wav, .m4a
  - Show selected files before upload
  - "Загрузить" button to start upload
  - Progress indicator per file: "Загрузка..." → "Обработка..." → "Готово"
  - Poll GET /api/kb/{kbId}/jobs/{jobId} every 2 seconds for status updates
  - Error state per file with error message
- Russian UI text, light theme, Tailwind CSS
- Use createSignal, createEffect, For, Show from solid-js
- Upload flow:
  1. POST /api/kb/{kbId}/upload with { fileName, contentType } → get { uploadUrl, sourceId, jobId }
  2. PUT uploadUrl (presigned R2 URL) with file body
  3. POST /api/kb/{kbId}/upload/confirm with { sourceId } → starts processing
  4. Poll job status until completed or failed
