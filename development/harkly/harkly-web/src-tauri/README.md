# Tauri 2 Desktop Shell (Stub)

Placeholder for Tauri 2 desktop build. Not active in MVP.

## When to activate

Phase 4 (Canvas + Polish) or v2 roadmap.

## Setup

1. Install Tauri CLI: `cargo install tauri-cli`
2. Copy from quantum template: `Cargo.toml`, `tauri.conf.json`, `lib.rs`
3. Set `TAURI_ENV_PLATFORM` env var
4. `app.config.ts` auto-switches to `preset: "static"` + `ssr: false`
