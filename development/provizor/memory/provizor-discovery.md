## Goal

Определить tech stack, архитектуру, получить доступ к данным Алимхана, провести базовые исследования. Результат: готовность к разработке Bot MVP (APT-02).

## Context

Проект "Provizor" — автоматизация сети из 5 аптек в Казахстане (партнёр Алимхан).
4 домена: Telegram-бот (Care as a Service), аналитика ассортимента, ценообразование, unit-экономика.
Хостинг на текущем Hetzner VPS рядом с Contexter.

## Tasks

### Research (R)

- [ ] R-01: Исследование Telegram Bot frameworks (aiogram vs grammY vs python-telegram-bot). Критерии: казахский STT, RAG интеграция, production readiness
- [ ] R-02: Исследование 1С интеграции — API, OData, COM, готовые коннекторы. Как получить ассортимент, цены, остатки, продажи
- [ ] R-03: Исследование Kaspi Pay API — подключение, комиссии, требования (ИП/ТОО), документация
- [ ] R-04: Исследование iTeka — есть ли API, структура данных, возможность парсинга цен конкурентов
- [ ] R-05: Исследование Groq Whisper — поддержка казахского языка, качество распознавания пенсионерской речи
- [ ] R-06: Исследование госреестра предельных цен РК — формат данных, API, обновления
- [ ] R-07: Исследование автоматизации ревизии — RFID/NFC маячки для аптек, существующие решения, стоимость
- [ ] R-08: Статистика "60% сначала в аптеку" — верификация через исследования
- [ ] R-09: Исследование аптечных SaaS в СНГ/Казахстане — конкуренты, что есть на рынке
