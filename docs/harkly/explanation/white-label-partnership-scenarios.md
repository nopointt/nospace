## Контекст

Head of Product Proxymarket увидел Harkly Desktop, передал собственникам. Собственники задали вопросы, фидбек от Артёма: **ждут предложения**. Цель: получить деньги после первого созвона, сохранить полное владение продуктом.

## Сценарий 1 — Рекомендованный: Paid Pilot → Годовая лицензия

### Финансовая схема

| Этап | Сумма | Условие |
|---|---|---|
| Пилот (2 месяца) | $1,500–3,000 фикс | 100% оплата при подписании |
| Годовой white-label контракт | $6,000–12,000/год | Подписание в течение 30–60 дней после пилота |
| Зачёт пилота | Стоимость пилота → скидка в первый год | Конверсионный механизм |

**Scope пилота:**
- Интеграция Harkly под брендом Proxymarket
- 1–2 реальных клиента Proxymarket на платформе
- Layer 1: мониторинг отзывов (WB/Ozon/Отзовик)
- Базовый AI-анализ: тональность, темы, претензии
- Итоговый отчёт для собственников

### Юридическая схема

Два коротких документа:

**Документ 1: Pilot Agreement (1 стр.)**
```
Обязательные пункты:
✅ IP принадлежит [nopoint / ИП / ООО] полностью
✅ Proxymarket получает право использования только в рамках пилота
✅ Без права передачи третьим лицам
✅ SLA: best effort на пилоте
✅ Оплата: 100% предоплата, до начала работ
✅ Результат пилота: описан конкретно (что сдаётся)
```

**Документ 2: Letter of Intent (0.5 стр.)**
```
Фиксирует намерение перейти к годовому контракту:
✅ Ориентировочные условия годового white-label
✅ Срок принятия решения (30–60 дней после пилота)
✅ Подтверждение, что LOI не является обязывающим по оплате
```

### Почему win-win
- **Proxymarket:** низкий риск, понятный outcome, деньги маленькие
- **Harkly:** кэш сразу, IP защищён, ясный путь к крупному контракту

## Сценарий 2 — Быстрее, меньше кэша: Discovery-спринт

### Финансовая схема

| Этап | Сумма | Условие |
|---|---|---|
| Discovery-спринт (3–4 недели) | $800–1,500 фикс | 100% оплата вперёд |
| Результат спринта | Кастомный прототип дашборда + роадмап | Передаётся в конце спринта |
| Дальше | Годовой лицензионный договор | По желанию сторон |

**Что входит:**
- Кастомный прототип дашборда под брендом Proxymarket
- Роадмап внедрения Harkly для их клиентской базы
- Список интеграций и технических требований
- Презентация для собственников

### Юридическая схема

Один документ: **Service Agreement (1–1.5 стр.)**
```
Обязательные пункты:
✅ «Все IP на инструмент и технологию принадлежат [nopoint]»
✅ «Proxymarket получает право использования результатов спринта и продукта
    в рамках будущего лицензионного договора»
✅ Оплата: предоплата до старта
✅ Описание deliverables (что именно сдаётся)
✅ Сроки
```

### Почему win-win
- **Proxymarket:** минимальный порог входа, быстро получают понятный артефакт
- **Harkly:** деньги после подписания, минимальные трудозатраты, IP защищён

## Сценарий 3 — Компромисс: Flat fee + Rev-share

> Использовать только если Сценарии 1 и 2 не прошли («аванс неудобен»)

### Финансовая схема

| Компонент | Сумма |
|---|---|
| Flat-fee (ежемесячно) | $300–500/мес |
| Rev-share | 25–35% от выручки Proxymarket с этого продукта |
| Minimum commitment | $300/мес гарантировано, независимо от продаж |
| Срок договора | Минимум 12 месяцев |

### Юридическая схема

**White-Label Reseller Agreement (~3–5 пунктов):**
```
✅ Лицензия на брендинг: неисключительная, ограниченная по географии/отрасли
✅ IP у nopoint
✅ Ежемесячная отчётность по выручке
✅ Право на аудит выручки (1 раз в год)
✅ Minimum commitment (защита от нулевых платежей)
✅ Условия расторжения: прекращение использования брендинга и технологии
```

### Почему win-win
- **Proxymarket:** низкий барьер входа, платят пропорционально успеху
- **Harkly:** гарантированный минимум + потенциально высокий upside

## Обязательные условия во всех сценариях

Независимо от выбранной схемы, в любом документе:

| Пункт | Формулировка |
|---|---|
| **IP** | «Все права на технологию, код и торговую марку Harkly принадлежат [nopoint]» |
| **Лицензия** | «Proxymarket получает неисключительную лицензию на использование и ребрендинг» |
| **Запрет передачи** | «Лицензия не может быть передана третьим лицам без письменного согласия» |
| **Данные** | «Данные конечных клиентов обрабатываются согласно политике конфиденциальности» |
| **Расторжение** | «При расторжении Proxymarket прекращает использование брендинга и технологии» |
| **Оплата** | «Оплата производится до начала работ / до начала периода» |
