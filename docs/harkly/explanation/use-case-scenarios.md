## UC-01 — E-commerce бренд: непрерывный мониторинг отзывов {#uc-01}

### Кто

**Катя, бренд-менеджер.** Управляет брендом одежды на WB + Ozon, команда 3 человека. Самостоятельно следит за репутацией, раз в неделю вручную просматривает отзывы.

### Проблема / Триггер

- Тратит 3–4 часа в неделю на ручной просмотр отзывов на нескольких площадках
- Узнаёт о волне негатива через 2–3 недели — когда рейтинг уже упал
- Не видит цельную картину: что конкретно бесит покупателей и по каким темам
- Не знает, как обстоят дела у конкурентов

### Путь в Harkly

```
1. Регистрация → создать проект «Мой бренд»
2. Добавить источники: WB + Ozon + Отзовик (3 клика)
3. Запустить первый парсинг → ждать 5 минут
4. Получить AI-анализ: тональность 72% позитив,
   топ-претензия — «размерная сетка» (18% отзывов)
5. Создать проект «Конкурент А» → сравнение бок о бок
6. Настроить алёрт: «если тональность упадёт на 15% — уведомление в Telegram»
7. Каждую неделю: 5 минут на дашборд вместо 4 часов
```

### Используемые слои

- **Reality Layer** — сбор + AI-анализ отзывов
- **Reality Layer** — конкурентная разведка

### Ключевые выходы

| Выход | Описание |
|---|---|
| Дашборд тональности | Динамика позитив/нейтрал/негатив за 90 дней |
| Топ-10 тем | Доставка, качество, размерная сетка, упаковка... с весами |
| Топ-претензии | «Размер не совпадает» — 18%, «доставка задержалась» — 12%... |
| Сравнение с конкурентом | NPS-proxy: мой бренд 68, конкурент А 74 → gap понятен |
| Алёрт | Telegram: «Внимание: тональность упала на 20% за 7 дней» |
| PDF-отчёт | Еженедельный branded-отчёт для руководителя |

### Метрика успеха

- Время на мониторинг: с 4 часов/нед → 15 минут
- Обнаружение негатива: с 2–3 недель → до 6 часов
- Первый инсайт на действие: в первый день после подключения

### Тариф

**Starter $49/мес** → при добавлении второго бренда/конкурента: **Growth $149/мес**

## UC-02 — Product Manager: прогноз реакции до запуска {#uc-02}

### Кто

**Антон, PM в SaaS-компании.** Запускает обновление карточки продукта и рекламного оффера. Команда потратила 2 недели на написание нового текста. Хочет проверить до запуска.

### Проблема / Триггер

- A/B-тест на реальной аудитории займёт 3–4 недели и стоит дорого
- Нет быстрого способа понять: «наш новый текст — улучшение или нейтральное изменение?»
- Прошлый оффер получил волну негатива из-за неудачной формулировки — хочет избежать повторения
- Надо защитить решение перед директором с данными, а не интуицией

### Путь в Harkly

```
1. Открыть вкладку «Predict»
2. Вставить текст нового оффера (300 слов)
3. Выбрать: аудитория = покупатели SaaS B2B, RU/СНГ
4. Нажать «Предсказать реакцию» → ждать 25 секунд
5. Результат: 58% позитив, 28% нейтрал, 14% негатив
   Риск: «непонятная ценовая модель» у 22% синтетических респондентов
6. Переписать раздел с ценовой моделью
7. Запустить повторный тест нового варианта
8. Второй результат: 71% позитив, риски снизились до 8%
9. Пойти на запуск с данными в руках
```

### Используемые слои

- **Prediction Layer** — silicon sampling, ensemble prompting
- **Reality Layer** — как baseline: реальные отзывы прошлых версий оффера для калибровки

### Ключевые выходы

| Выход | Описание |
|---|---|
| Predicted sentiment | 58% / 28% / 14% — с confidence 71% |
| Predicted topics | Топ-5 вероятных тем реакции с направлением (позитив/негатив) |
| Risk flags | Конкретные риски с вероятностью: «22% упомянут ценовую путаницу» |
| Calibration note | «Откалибровано на 980 реальных отзывах. Точность ±12 п.п.» |
| Before/After | Сравнение двух версий оффера рядом |

### Метрика успеха

- Цикл тестирования оффера: с 3–4 недель → 30 минут
- Количество вариантов до запуска: 1–2 → 5–10 (быстрее и дешевле)
- Снижение postlaunch-негатива: цель — на 30%+

### Тариф

**Growth $149/мес** (50 prediction-запросов) → **Pro $299/мес** (безлимит prediction)

## UC-03 — Маркетинговое агентство: white-label для клиентов {#uc-03}

### Кто

**Игорь, CEO маркетингового агентства.** 12 клиентов — FMCG, ритейл, e-commerce. Продаёт репутационный мониторинг, но делает всё руками или через дорогой YouScan. Хочет поднять маржу.

### Проблема / Триггер

- YouScan стоит 80 тыс. ₽/мес — съедает маржу, клиентов надо много чтобы окупить
- Ручной мониторинг занимает 20+ часов в неделю у команды
- Нет инструмента для красивых branded-отчётов под логотипом агентства
- Клиенты просят «предсказательную аналитику» — пока нечего предложить

### Путь в Harkly

```
1. Получить Enterprise-лицензию
2. Настроить white-label: логотип агентства, цвета, домен analytics.agency.ru
3. Создать 12 проектов — по одному на каждого клиента
4. Настроить субаккаунты: каждый клиент видит только свои данные
5. Для клиента А: подключить WB + Яндекс.Карты + Отзовик
6. Клиент А заходит на analytics.agency.ru — видит «свой» инструмент
7. Раз в месяц: экспортировать branded PDF-отчёт → отправить клиенту
8. Для ключевых клиентов: добавить Prediction Layer
   «Прежде чем запускать акцию — тестируем текст в нашем инструменте»
9. Поднять чек с 30 тыс. → 55 тыс. ₽/мес за «умный мониторинг»
```

### Используемые слои

- **Reality Layer** — мониторинг для всех 12 клиентов
- **Prediction Layer** — как premium-услуга для ключевых клиентов
- **White-label** — полный ребрендинг под агентство

### Ключевые выходы

| Выход | Описание |
|---|---|
| Мультиклиентский кабинет | 12 клиентов изолированы, агентство видит всех |
| Branded дашборд | analytics.agency.ru с логотипом агентства |
| Branded PDF-отчёты | С логотипом агентства, не «Powered by Harkly» |
| API-доступ | Интеграция с CRM агентства или собственной отчётностью |
| Prediction как сервис | Дополнительная услуга «тестирование оффера» |

### Метрика успеха

- Экономия на YouScan: $1,000/мес → $499/мес (Enterprise) — маржа выросла
- Время на ручной мониторинг: 20 часов/нед → 2 часа
- Churn клиентов: снижение, так как теперь есть sticky-инструмент
- Средний чек клиента: +$250/мес за «умный мониторинг»

### Тариф

**Enterprise $499/мес** (white-label + API + мультиклиент)

## UC-04 — Платформенный партнёр: ProxyMarket white-label {#uc-04}

### Кто

**ProxyMarket** — B2B-инфраструктурная платформа. Продаёт прокси для сбора данных крупным брендам и агентствам. Клиенты уже используют прокси для парсинга, но аналитики поверх данных нет.

### Проблема / Триггер

- Клиенты платят за прокси, но не получают аналитику — уходят к тем, кто даёт готовые insights
- ProxyMarket хочет повысить LTV и дифференцироваться от commodity-прокси-провайдеров
- Строить собственный аналитический продукт с нуля — не core и стоит $200k+
- У ProxyMarket есть отзывы на 101poisk, Яндекс.Картах, Yell, Trustpilot — но нет их собственной аналитики

### Путь в Harkly

```
Пилот (2 месяца):
1. Подписать Pilot Agreement → $2,000 предоплата
   + получить Enterprise бесплатно на срок пилота
2. Harkly настраивает white-label под ProxyMarket:
   analytics.proxymarket.ru, их логотип и цвета
3. ProxyMarket подключает 2 пилотных клиента
4. Пилотный клиент 1: бренд одежды — мониторинг WB/Ozon
5. Пилотный клиент 2: B2B-SaaS — мониторинг Trustpilot/G2
6. Параллельно: ProxyMarket использует Enterprise-аккаунт
   для собственной аналитики (свои отзывы + конкуренты)
7. Через 2 месяца: отчёт для собственников, переход к годовому WL
```

```
Полный WL (после пилота):
1. Годовой контракт $8,000/год
2. «ProxyMarket Analytics» — продукт под их брендом
3. Клиенты ProxyMarket подключают свои аккаунты
4. ProxyMarket добавляет аналитику как upsell к прокси-тарифам
   «Прокси + Аналитика: +$99/мес»
5. Revenue share от Harkly начинается с N-го клиента
```

### Используемые слои

- **Reality Layer** — мониторинг для клиентов ProxyMarket
- **Prediction Layer** — premium-фича для enterprise-клиентов партнёра
- **AI Perception Layer** — мониторинг LLM-нарратива (уникальный дифференциатор)
- **White-label** — полный ребрендинг под ProxyMarket

### Ключевые выходы

| Выход | Описание |
|---|---|
| analytics.proxymarket.ru | Готовый аналитический продукт под их брендом |
| Собственная аналитика ProxyMarket | Отзывы на 101poisk, Yell, Trustpilot, G2 — все в одном дашборде |
| Upsell-инструмент | «Прокси + Аналитика» как новый тарифный блок |
| Клиентские отчёты | Branded PDF под ProxyMarket для их корпоративных клиентов |
| LLM-аудит | Как ChatGPT/YandexGPT/GigaChat описывает ProxyMarket vs конкурентов |

### Метрика успеха

- Первые деньги: $2,000 после подписания пилота
- LTV клиента ProxyMarket: рост на 20–30% за счёт upsell
- ProxyMarket MRR от аналитики: 10+ клиентов × $99/мес = $990+/мес → растёт

### Тариф

**Partnership: Paid Pilot $2,000 → Annual WL $8,000/год** (см. [Сценарий 1](./partnership-scenarios.md))

## UC-05 — Mid-market бренд: полный цикл UX-интеллидженса {#uc-05}

### Кто

**Дмитрий, директор по маркетингу** в компании 200+ человек (инфраструктурный SaaS, RU/СНГ + EU). Есть YouScan для PR, но нет product/UX-инструмента. Хочет понимать, как клиенты воспринимают продукт и бренд — во всех точках.

### Проблема / Триггер

- YouScan покрывает соцсети, но не нишевые площадки (G2, Trustpilot, тематические форумы)
- Нет способа быстро протестировать новый positioning и key messages до запуска кампании
- Совет директоров спросил: «А как нас описывает ChatGPT / Яндекс?» — ответа нет
- SEO-агентство делает GEO-аудит раз в квартал за $3K — хочет self-service

### Путь в Harkly

```
Reality Layer (Неделя 1–2):
1. Подключить бренд: G2, Trustpilot, Яндекс.Карты, профильный форум
2. Получить AI-анализ: что хвалят, что критикуют, топ-темы
3. Настроить мониторинг конкурентов (2–3 бренда)
4. Настроить алёрты в Telegram для команды

Prediction Layer (Неделя 3–4):
5. Перед запуском нового positioning — тест на Prediction
6. «Наш новый нарратив: надёжная инфраструктура для enterprise»
   → Prediction: 68% позитив, риск «воспринимается как дорогой» у 24%
7. Откорректировать messaging, запустить

AI Perception Layer (Месяц 2):
8. Запустить мониторинг LLM: ChatGPT + YandexGPT + GigaChat
9. Результат: ChatGPT упоминает бренд в 40% ответов по теме,
   YandexGPT — в 12%. Gap обнаружен.
10. Action plan: публикации на Habr, VC, профильных ресурсах
11. Через месяц: YandexGPT-присутствие выросло до 28%
```

### Используемые слои

- **Reality Layer** — мониторинг нишевых B2B-площадок + конкурентная разведка
- **Prediction Layer** — тест positioning и ключевых сообщений
- **AI Perception Layer** — мониторинг и управление LLM-нарративом

### Ключевые выходы

| Выход | Описание |
|---|---|
| Multi-platform дашборд | G2, Trustpilot, Яндекс.Карты, форумы — одним экраном |
| Competitive UX-benchmark | Мой бренд vs 3 конкурента: NPS-proxy, топ-темы, тренды |
| Positioning test | Новый нарратив протестирован до запуска → данные для совета директоров |
| LLM Brand Report | ChatGPT/YandexGPT/GigaChat — presence rate, нарратив, изменения за квартал |
| Action plan | Конкретные рекомендации: где публиковаться, что исправить в продукте |

### Метрика успеха

- GEO-аудит: с $3K/квартал (агентство) → $299/мес (self-service)
- Positioning-цикл: с 4 недель A/B → 1 день prediction
- LLM-присутствие YandexGPT: рост с 12% → 28% за месяц действий
- Совет директоров: есть данные о том, «как нас видит ИИ»

### Тариф

**Pro $299/мес** → при добавлении white-label или API-интеграции: **Enterprise $499/мес**
