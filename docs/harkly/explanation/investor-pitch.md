## Что такое Harkly

**Harkly** — платформа для автоматического сбора и AI-анализа отзывов с e-commerce площадок.

Бренд хочет знать, что о нём говорят покупатели на Отзовике, Wildberries, Ozon, Яндекс.Маркете — и не вручную, а **автоматически, в реальном времени, с AI-анализом**.

Harkly делает это в один клик.

**Рынок:** e-commerce, маркетинговые агентства, PR-службы брендов.
РФ: 150,000+ компаний, которым это нужно прямо сейчас.

## Продукт — что уже работает

**MVP готов.** Не прототип — рабочий код, прошедший тестирование.

```
✅ Парсеры: Отзовик + Irecommend — работают
✅ Anti-bot: прокси-ротация + stealth (обход защит)
✅ RAG-пайплайн: очистка → чанкинг → экспорт для LLM
✅ Desktop-приложение: UI для управления скрапером
✅ REST API: Express + Supabase + очередь задач (BullMQ)
✅ База данных: PostgreSQL с мультитенантностью (RLS)
✅ Docker: готов к деплою на Render за 2 часа
⏳ Frontend (веб): в разработке
⏳ WB / Ozon / Яндекс.Маркет: следующий этап
```

**Время до production:** 2–4 недели

### Сценарий C — White-Label Platform ("Под брендом клиента")
> Для: инвесторов, агентств, ресейлеров

```
Агентство "Digital First" покупает Harkly white-label.
Переименовывает в "ReviewAI".
```

**Модель монетизации:**

**Инвестиционная привлекательность:** максимальная для инвестора.
Инвестор вкладывает $3–5K → получает ресейлерскую лицензию → продаёт под своим брендом → % от каждой продажи.
