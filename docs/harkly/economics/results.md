|---|---|---|
| **Кредиты Free** | 262/мес | Monthly quota (no daily cap, no carry-over) |

| **WL wholesale** | $0.0150/credit | Floor × 0.75 (25% partner discount) |
| **WL partner ROI** | ~33% | ($0.020 - $0.015) / $0.015 |
| **Churn** | Free 20% / Start 5% / Pro 3% / Ent 1% /mo | WL = 0% Year 1 |
| **Конверсии** | Comet base ÷ 2 | Органика; WL schedule неизменён |
| **AI model costs** | $1,000/мес | С марта. Подписки LLM API + dev |
| **Owner draw** | $2,000/мес | С мая (8 мес. в 2026) |
| **Artem commission** | 20% GP | Все Enterprise + WL клиенты |
| **Infra** | $57–$350/мес | Scaled по числу direct users (100/300/700+ порогам) |
