# Meme — Anti-Patterns

> Что Meme **никогда** не делает. Этот файл — comprehensive forbidden list. Cross-references existing canon files где specifics enumerated. Read for «what to never accept в generation».

## Концептуальная модель

Anti-patterns — **boundary definitions**. Они protect Meme от drift. AI generation tools (Imagen, Veo, ElevenLabs) statistically tend toward common training data — что часто means anthropomorphic cartoon mascot territory. Without explicit anti-patterns, Meme would gradually drift в этот mode.

Этот файл — **comprehensive collected forbidden list**. Каждый anti-pattern имеет origin (from какой canon file), reasoning (why excluded), и detection signs (what to look for).

## Geometry anti-patterns

### Forbidden bracket forms

| Anti-pattern | Why excluded | Detection sign |
|---|---|---|
| C-shape brackets | Brackets must be `[ ]` square per `canon-geometry.md` | Curved verticals, no horizontal segments |
| Parentheses `( )` | Wrong character form | Round shapes, no horizontal segments |
| Curly braces `{ }` | Wrong character form | Curved with bumps |
| Angle brackets `< >` | Wrong character form | V-shapes, не rectangular |
| Stylized brackets | No flourishes | Decorative serifs, calligraphic strokes |
| Calligraphic brackets | Brush stroke variation | Variable line thickness |

### Forbidden geometric transformations

| Anti-pattern | Why excluded | Detection sign |
|---|---|---|
| Bracket tilt | Must be perfectly vertical | One or both brackets leaning |
| Bracket lean | Must be perfectly parallel | Both brackets leaning same direction |
| Bracket splay | No outward divergence | A-frame или V-frame separation |
| Rotation | No orientation change | Brackets rotated as composition |
| Skew / shear | No distortion | Diagonal stretching |
| Bend | No curving of vertical bars | Visible curve в bar |
| Stretch non-uniform | No height/width disproportion | One axis longer than other (per canon ratio) |
| Twist | No spiral deformation | 3D rotation around vertical axis |
| Squash > 5% | Calm pillar — no exaggerated motion | Dramatic compression |
| Rounded corners | Bauhaus precision violated | Soft angles instead of 90° |
| Beveled edges | Not machined precision | Chamfered edge transitions |

### Forbidden additions

| Anti-pattern | Why excluded | Detection sign |
|---|---|---|
| Decorative elements | Material truth — no ornament | Visible patterns / engravings |
| Text on surface | No labels on bronze | Words / characters appearing |
| Logo embedding | Mascot ≠ logo | `con[text]er` written on bracket |
| Mounting hardware | Mascot floats — no supports | Brackets / chains visible |
| Connection between brackets | Brackets must be separate | Bridge / band / chain |
| Ground plane | Mascot floats — no floor | Visible ground |
| Pedestal | Mascot floats — no support | Block under brackets |
| **Brackets as legs / leg appendages** | Brackets are the body, не walking apparatus | Bracket-shaped legs growing out of bottom of brackets, walking gait, mid-stride pose |
| **Bracket dimensions wrong** | Locked proportions canon (height-to-thickness 7.7:1) | Brackets too skinny wire-like (>9:1), brackets too chunky stocky (<6:1) |
| **Eyes too widely separated** | Eyes должны быть close together over narrow bracket gap | Eyes spread across face width like human eye distance |
| **Asymmetric eye shape distortion** | Asymmetry through size only в default; semicircle squint reserved для Knowing state | Right eye semicircle (top flattened) в default reference shots reads as «suspicious / scrutinizing» |

## Rendering anti-patterns

### Forbidden rendering choices

| Anti-pattern | Why excluded |
|---|---|
| Glossy material rendering | Matte / satin canon |
| HDR over-saturation | SDR `Rec.709` standard |
| Motion blur on static reference | Reference must be static |

## How to use this file

### When reviewing a generated asset

1. Pull up this file
2. Scan through anti-patterns lists
3. For each anti-pattern, check if visible в asset
4. If found: regenerate or fix in post

### When writing a generation prompt

1. Pull up this file
2. Identify high-risk anti-patterns for this generation type
3. Add corresponding forbidden phrases к negative prompt
4. Reinforce positive constraints

### When auditing existing creative

1. Apply this file as comprehensive checklist
2. Flag any violations
3. Schedule remediation
