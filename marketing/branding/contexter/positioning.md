## Одна фраза

Контекстер — слой контекста, в котором ваша AI-память принадлежит вам и читается любой моделью.

## Для кого

AI-power user, 28-50 лет. Использует 3+ модели в день — Claude, ChatGPT, Gemini, Perplexity, Mistral. Может быть аналитик, исследователь, PM, founder, писатель, преподаватель, разработчик. Объединяет их одно: **они уже устали перетаскивать контекст между моделями руками**.

Это не абстрактный «pro user». Это человек, у которого:
- В Claude Project — 12 PDF к проекту X.
- В ChatGPT Custom GPT — стиль письма + любимые статьи.
- В NotebookLM — 30 источников к статье в работе.
- В Perplexity Spaces — браузерная история.
- На рабочем компьютере — папка `Research 2025/` с 200 файлами.

И это всё про одну работу. Но контекст разорван по платформам, и каждый разговор начинается с того, что «вспомнить» нужно вручную.

## Проблема которую решаем

**Человеческий контекст становится самым ценным ресурсом в эпоху AI.**

Алгоритмы воспроизводимы. Веса моделей копируемы. Open-source RAG-стек собирается за выходные. То что человек прочитал, обдумал, написал в течение месяцев и лет — невосстановимо извне.

И именно эта самая ценная часть рассыпана по сервисам, которые не разговаривают друг с другом. Каждая модель строит свою клетку памяти и пытается поместить туда пользователя.

Это не ошибка отдельного продукта. Это **структурный недостаток рынка**: AI-вендоры экономически заинтересованы привязать пользователя к своей модели через память. Память становится lock-in.

## Наш ответ

Контекстер — единственный архив, который читает любая AI.

Файлы хранятся в R2 под контролем пользователя. Вектора в pgvector. Доступ через MCP / REST / прямой query.

**Public RAG** — открытые базы знаний с управляемым доступом. **Shared Rooms** — общие пространства контекста с правами. **Any size, any format** — приёмник всего, без подгонки.

Это не «AI с лучшим RAG». Это **другой слой реальности**: слой, на котором память принадлежит человеку, а модель — её гость.

## Почему мы другие

- **Pinecone / Weaviate / Qdrant** — это инфраструктура для разработчиков. Контекстер — продукт для конечного пользователя. (Хотя API есть.)
- **NotebookLM / Claude Projects / Custom GPTs** — это память внутри одного вендора. Контекстер — память поверх всех вендоров.
- **Notion / Roam / Obsidian** — это написать. Контекстер — это **сохранить и вспомнить через AI**, без необходимости писать.
- **Mem.ai / Reflect** — это AI-aware notes. Контекстер не делает заметки. Он принимает то что у вас уже есть, в любом формате, и делает читаемым любой моделью.

Дифференциатор не в фичах. Это **позиция**: вы — владелец вашего контекста. Модель — гость, не хозяин.

## Что мы НЕ делаем (сейчас)

- **Не делаем enterprise-RAG.** Фокус на индивидуальных пользователях и небольших командах через shared rooms. Корпоративный сегмент — потом.
- **Не делаем chat-интерфейс.** Контекст отдаётся в Claude / ChatGPT / Gemini / любую — там и идёт диалог. Контекстер — слой памяти, не UI чата.
- **Не делаем training.** Никакого silent fine-tuning на пользовательских данных. Только storage + retrieval.
- **Не делаем агентов.** В этой версии. Контекстер — субстрат для агентов, не агент.
- **Не делаем mobile native app.** Web app + API. Mobile через web. Native — позже.
- **Не делаем on-prem deployment.** SaaS. Open-source / self-host рассмотрим после PMF.

## GTM — как мы растём

**Контентный путь.**

Первые 6-12 месяцев — не продажи, а формирование категории. Контекстер публично формулирует и называет «Public RAG + Shared Rooms + Any-format» как новую категорию: статьи, разборы, кейсы, snippets через 11 платформ (HN / Reddit / Twitter / LinkedIn / Telegram / Substack / Dev.to / Medium / Hashnode / IndieHackers + blog.contexter.cc).

Каждый день — 2 дайджеста + 2-3 статьи. Master article на blog.contexter.cc — оригинал. На каждой целевой платформе — отдельная адаптация под аудиторию (D-CTX15-05).

**Канал-якорь:** blog.contexter.cc + Hacker News (через master articles + Show HN). Telegram — для русскоязычной аудитории.

**GTM-01 Omnipresence epic** управляет первым контактом со всеми 28 touchpoints (4 продукта × Tier 1 платформы) — см. `nospace/memory/gtm-01-omnipresence-epic.md`.

## Метрика успеха (следующие 6 месяцев)

Категория-лидер — это измеримо:

- Термины «Public RAG», «Shared Rooms», «Sovereign Context» используются в постах третьих лиц без нашей инициативы.
- Контекстер упоминается в HN-тредах + Reddit-тредах про AI memory / RAG / multi-model workflows как референс.
- 100 пользователей supporters program ($10) перешли в платных subscribers ($9 Starter / $29 Pro) с retention 50%+ через 30 дней.
- **Human Context Sovereignty** становится темой разговора — не «у вас есть RAG в продукте?», а «как вы решаете проблему контекстного lock-in?».

Не количество подписчиков. Не объём данных в архиве. **Признание категории как проблемного поля.**
