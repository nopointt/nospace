# Principles — Contexter Adaptation

Five Bauhaus principles. Universal structure, cold application.

Source: Harkly `foundations/principles.md` (universal). RAG: `bauhaus_knowledge` collection.

---

## 1. Gestaltung — Form Through Operation

**Rule:** Form emerges from structural role, not from physical-world analogy.

**Contexter adaptation:** A file upload zone is not a "mailbox." A pipeline indicator is not a "conveyor belt." Form follows data flow and functional role.

**Sharp corners required.** Mondrian P-37: "Let every visible relationship be governed by the right angle." Contexter's 0px radius is a formal choice expressing precision, not aesthetic preference.

**Check:** Can the element's form be explained without "it looks like a..."?

---

## 2. Gleichgewicht — Dynamic Equilibrium

**Rule:** Achieve tensioned equilibrium through asymmetric balance, never through static symmetry.

**Contexter adaptation:** The 8:4 column split (hero text left, upload right). Pipeline stages progress left→right with increasing visual weight. Data tables use asymmetric column widths by content type.

Mondrian P-17: "Without symmetry — through all things equilibrium must be created."

**Check:** Remove any element. Does the balance shift? If not, the element is decorative.

---

## 3. Materialwahrheit — Material Truth

**Rule:** Use every material according to its inner law. Screen = light + geometry + typography.

**Contexter adaptation:** The screen is a light-emitting surface. Contexter's material law is **cold precision**:
- Pure white (#FAFAFA) = honest use of screen luminance without warm tinting
- Pure black (#0A0A0A) = maximum contrast, no softening
- No shadows whatsoever — elevation through border and background shift only
- No gradients, no textures, no blurs

**Check:** Does any element simulate a physical material? Does any element have a shadow? Remove it.

---

## 4. Sparsamkeit — Economy of Means

**Rule:** Use the minimum elements necessary for maximum clarity.

**Contexter adaptation:** This is the defining principle.
- 3 colors (black, white, blue) — Gropius: "minimum means, maximum effect"
- 1 typeface (JetBrains Mono) — Zeitschrift: "single-typeface"
- 0px corners — Van Doesburg: "pure plastic means"
- No icons where text suffices
- No borders where space separates
- Data IS the interface — no chrome between user and data

**Check:** Remove the element. Does comprehension decrease? If not, remove it.

---

## 5. Raumgestaltung — Space as Active Participant

**Rule:** Space is not a container for elements. Space IS an element — active, weighted, compositional.

**Contexter adaptation:** White space is the primary compositional tool. The spacing scale (4px base, 0-80px) encodes intentional rhythm. Contexter uses generous white space — not as emptiness, but as Swiss precision breathing.

Zeitschrift: "Inter-word spacing is narrow. The overall texture of text pages is even, with no rivers of white. Uniform typographic colour."

Klee P-17: Proportions as "specific ratios between unlike quantities — 2:3:5."

**Check:** Can you describe the spatial rhythm as an intentional pattern? If spacing is uniform everywhere, the space has not been designed.
